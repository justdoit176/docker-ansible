# docker-ansible
Docker image for Ansible software, based on Alpine.

Versions:
- Alpine 3.10
- Python 3.7
- Ansible 2.8.6

Usage:
`docker pull armox76/ansible`

[Docker Image](https://hub.docker.com/r/armox76/ansible/)
