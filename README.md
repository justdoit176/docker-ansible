# docker-ansible
Docker image for Ansible software, based on Alpine.

Versions:
- Alpine 3.12
- Python 3.8.5
- Ansible 2.10.4

Usage:
`docker pull armox76/ansible`

[Docker Hub](https://hub.docker.com/r/armox76/ansible/)
